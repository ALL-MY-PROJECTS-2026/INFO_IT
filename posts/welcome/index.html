INFO IT 블로그를 시작하며

Jungwoo2026년 7월 22일
#소개#시작하기

안녕하세요. INFO IT는 개발과 IT 지식을 실무 경험과 함께 정리하는 기술 블로그입니다. 이 첫 글에서는 블로그의 방향과 이 사이트를 만든 기술 스택을 간단히 소개합니다.

왜 이 블로그를 만들었나

정보는 넘치지만, 정작 "직접 해보며 깨달은 맥락"이 담긴 글은 드뭅니다. 이 블로그는 단순 요약이 아니라 다음을 지향합니다.

  • 실제로 부딪힌 문제와 해결 과정
  • 복사해서 바로 쓸 수 있는 코드 예시
  • 왜 그렇게 동작하는지에 대한 배경 설명

어떤 주제를 다루나

  1. 웹 프론트엔드 / 백엔드
  2. 클라우드 · 인프라 · DevOps
  3. 컴퓨터 과학 기초

이 사이트의 기술 스택

이 블로그 자체도 하나의 예제입니다. 다음 스택으로 만들어졌습니다.

// Vite + React + TypeScript + SSG(정적 사이트 생성)
export const stack = {
  build: 'Vite',
  ui: 'React 18 + TypeScript',
  content: 'MDX',
  render: 'vite-react-ssg', // 빌드 타임 프리렌더로 SEO 대응
  highlight: 'Shiki',
}

정적 프리렌더(SSG)를 사용해 검색 엔진과 광고 크롤러가 완성된 HTML을 받도록 설계했습니다. 앞으로 이런 선택의 이유들을 하나씩 글로 풀어가겠습니다.

피드백은 언제나 환영합니다. 함께 배워가요!
